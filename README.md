# younjunho17-web.github.io

Personal site for [@younjunho17-web](https://github.com/younjunho17-web).

Built with vanilla HTML/CSS/JS on the **leperche** design system —
warm paper surfaces, organic color fields, editorial typography.

## Local preview

```bash
cd ~/younjunho17.github.io
python3 -m http.server 8000
# → http://localhost:8000
```

## Deploy

This repo is published via GitHub Pages from the `main` branch.
Pushing to `main` redeploys at https://younjunho17-web.github.io.
